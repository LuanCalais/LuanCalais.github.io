Instant Chat - Chat em Tempo Real

Aplicação de chat em tempo real com Socket.IO. Stack MERN (MongoDB, Express, React, Node.js) com autenticação JWT via cookies httpOnly, upload de imagens (Cloudinary), state management com Zustand, e UI responsiva com TailwindCSS + DaisyUI. Dashboard com status online/offline e suporte a múltiplos temas.

React Node.js Socket.IO MongoDB Zustand TailwindCSS JWT
GitHub

Chrods & Notes - Gerenciamento de Músicas e Artistas

Sistema para gerenciamento de bandas e músicas, desenvolvido com MERN Stack (MongoDB, Express, React, Node.js) com integração de IA via Google Gemini. Features: dashboard analítico com visualizações interativas (Chart.js), CRUD completo de bandas e músicas, geração automática de conteúdo sobre músicas e bandas via IA (Gemini API), personalização de temas com color picker, autenticação segura com bcrypt e gerenciamento de estado global com Context API.

React Node.js MongoDB Chart.js Context API Gemini API
GitHub

Sistema fullstack para interação social focado em animais de estimação, desenvolvido com TDD. O frontend utiliza Vue.js, e o backend é construído com Java e Quarkus. Funcionalidades: feed de posts, perfis de pets, sistema de amizade e integração com PostgreSQL. Testes unitários com JUnit e deploy containerizado com Docker.
